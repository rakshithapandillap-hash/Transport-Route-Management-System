# Transport Route Management System

A full-stack web application developed to manage transportation operations in educational institutions.

## Features

* Dashboard with overview statistics
* Student Management (Add, Edit, Delete, Search)
* Route Management (Add, Edit, Delete, Search)
* Driver Management (Add, Edit, Delete, Search)
* Fee Management (Add, Delete, Search)
* MySQL database integration
* REST API using Express.js

## Technologies Used

### Frontend

* React.js
* Axios
* React Icons
* CSS

### Backend

* Node.js
* Express.js

### Database

* MySQL

## How to Run the Project

### Backend

1. Navigate to the backend folder.
2. Install dependencies:

npm install

3. Start the server:

npx nodemon server.js

### Frontend

1. Navigate to the frontend folder.
2. Install dependencies:

npm install

3. Start the application:

npm run dev
